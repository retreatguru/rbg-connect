== Description ==

This plugin is used to display the program catalog for the Retreat Booking Guru API.
Retreat Booking Guru is a retreat management software system for managing meditation, yoga and other retreat centers.
More information at [Retreat Booking Guru](https://retreat.guru/booking)

== Installation ==

1. Install plugin
1. Go to Retreat Booking Guru in the admin menu. Then go to plugin settings.
1. Enter the name of your subdomain. Ie. mycenter. Save.
1. To get the programs list to display properly in your site you may need to add html markup in "Before theme & after"
